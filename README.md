# Test Forms App

Небольшое тестовое приложение на React + TypeScript с использованием React Router DOM, Formik, Yup и Reactstrap.

## Описание

В приложении реализованы три формы:

- **Простая форма** (`NPSimpleForm`)
- **Форма с вкладками** (`NPTabbedForm`)
- **Многостраничная форма (Wizard)** (`NPWizard`)

Слева отображается вертикальное меню для переключения между формами, справа — выбранная тестовая форма.

Также реализованы тестовые компоненты:

- `NPTestForm1` — поле **Имя**
- `NPTestForm2` — поле **Фамилия**
- `NPTestForm3` — поле **Электронная почта**

Для работы с формами используется **Formik**, для валидации — **Yup**, для UI-компонентов — **Reactstrap** и **Bootstrap**.
